## Install Packages

```sh
pnpm i
```

## Init Grit

```sh
pnpm g:i
```

## Run Grit

### Show all patterns list

```sh
pnpm g:l
```

### Show only local patterns list

```sh
pnpm g:ll
```

### Apply pattern

```sh
pnpm g:ap 'pattern name'
```

### Run Test

```sh
pnpm g:t
```
